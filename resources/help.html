unitED Calculator Help

This is the general help page for the unitED calculator. This page contains useful information on the displayed components and how to use them. Hover over the faded text to get the full description!

Parts of the Calculator

Display Panel
At the top of the calculator is the display. This allows you to see the current expression so you remember what you have done so far and fix any mistakes.
Result Units Dropdown
Next to the display. The Result Units Dropdown allows you to determine the units to use to display results for calculations involving like-units. This allows you to work in units you are comfortable with.
Input Panel
Just below the display. This allows you to enter the current operand by typing on the physical keyboard or using the "soft keys"
Unit Dropdown
On the right of the Input is the Unit Dropdown, which is used to enter or select the units associated with the operands. This defaults to unitless (empty), but allows for properly formatted expressions and results if a unit is typed or selected.
Sign Button
The button labeled with the "plus-minus" symbol is used to toggle the sign of the current operand. This allows you to work more quickly.
Reset Button
By clicking on the button labeled "R" (for Reset), both the current expression and the current operand will be cleared. This allows you to restart the data entry process.
Clear Button
By clicking on the buttton labeled "C" (for Clear), the current operand will be cleared. This allows you to fix mistakes made when entering an operand.
Operation Buttons
The "soft" buttons in the display can be clicked to indicate the operator you want to use, after you have entered an operand in the input.
Equals Button
By clicking the button labeled "=", the expression you have entered will be evaluated
Number Pad
This gives users who prefer to type on-screen the ability to do so. The soft number buttons (including the sign) can be used to input operands.
Backspace Button
The button with the "backarrow" icon is the delete key, and can be used to delete the rightmost digit in the numeric part of the current operand. This allows for the correction of mistakes.
Integer Power Button
The button labeled "xy" is the integer power button. This can be used to calculate the integer power of an operand so that you can work with exponents more easily.
Inverse Button
The button labeled "1/x" is the inverse button. This can be used to invert an operand, which allows you to "reverse" an operation and/or invert units
History Panel
By clicking on the arrow bar on the right side of the window, the history panel will slide out to show the calculations you have performed so far. This allows you to reference previous expressions and fix mistakes.
Intermediate Steps Panel
By clicking on the arrow bar on the left side of the window, the intermediate steps panel will slide out to show steps used to get from the initial expression to the result so that you can teach people about working with units.

Entering Inputs

Operand Input
The system allows both the calculation of unitless and measured operands, so that a wider array of calculations can be performed.
Unit Input
The system supports selecing a unit from the input dropdown, or by typing your own! If typing your own, this supports multiple dash-conjunctions or slash/conjunctions so that you can use the system in a reasonably sophisticated way and solve any problems you are given.

System features

Running Calculations
The system allows for using the result of the previous expression in the next expression when you click an operator button.
Like Units
The system can perform operations on like units (such as miles and meters, or grams and ounces) that are predefined in the dropdown. This allows you to avoid having to perform conversions manually.
Define New Like Units
The system allows the user to define new like units and their associated conversion factors. This allows you to use the system with units it does not originally understand.
Result Unit Options
The system automatically picks the units in the current expression and populates the Result Unit Dropdown with them, allowing you to choose amongst reasonable options.
Physical Keyboard Operators
The system allows you to use the physical keyboard to enter oeprations as if you were using a soft key so that you can work more efficiently.
NOTE: Number pad operators work as expected, if present. In addition, the minus/underscore key acts as - (minus), the plus/equals key acts as + (plus), the x key acts as x (times), and the slash/question key acts as / (divide).
Physical Keyboard Digits
The system allows you to use the physical keyboard to enter digits as if you were using a soft key so that you can work more efficiently.
NOTE: The number pad numerals work as expected, if present. In addition, the top row numbers in the condensed keyboard layout work as expected.
System Languages
Depending on your locale, the system can be distributed in different languages. Currently included languages are English, French, and Russian.
Print Feature
The system allows the user to print the calculator history with the "File" button in the menubar.
